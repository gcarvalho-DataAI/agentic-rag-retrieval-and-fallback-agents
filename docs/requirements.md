# Requirements

## Functional

- Load and validate game JSON data with Pydantic
- Index games into a persistent vector database
- Query the vector database for semantic search
- Agent workflow that evaluates retrieval and falls back to web search
- Provide clear, structured answers with sources

## Non-Functional

- Python 3.11+
- Reproducible setup with `uv`
- CLI-first usage
- Automated tests and CI
